Elliptic Curve Cryptography

Elliptic Curve Cryptography (ECC) is an approach to public-key cryptography

It is based on the algebraic structure of elliptic curves defined by the formula:

\[\begin{aligned} \ y^{2} = x^{3} + ax + b \end{aligned} \]

Public-key cryptograpy is based in Public and Private keys

ECC public key is a point in the curve

ECC private key is an integer

Let's plot the following curve to explain step by step how to get to these keys

\[\begin{aligned} \ y^{2} = x^{3} - 3x - 5 \end{aligned} \]

This point P' is defined as

P' = A + B

Point P' becomes point A for the next iteration

This process is repeated N times

The final P' point is the public key

The times we have jumped to get to the final point is N, the private key

Let's try another approach of the same procedure

Since sometimes it's difficult to pick 2 different points in the curve

We can pick a single point P and pick 2 points extremely close to it

This way we can understand better how do we get to the private key

We will plot the same curve as before:

\[\begin{aligned} \ y^{2} = x^{3} - 3x - 5 \end{aligned} \]
